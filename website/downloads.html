MAR1D also has a Nix package, so you can simply install it through Nix if you are on Linux or Mac OS X.

Regarding the two linux downloads, the binary is a universal binary that should run on any distro, but is slow to start up, and also might have issues on AMD graphics cards. The flatpak is faster and more reliable, but requires flatpak installed on your system.
