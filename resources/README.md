MAR1D - /resources
===

``` sh
.
├── mario.font         # defines the font used in the game, character by character for the whole ascii character space
├── menuscreen_bg.png  # background for the main menu
├── selected.png       # icon to the left of the selected menu item
├── data               # the bulk of the game data. defines colours, objects, and levels. format can be gleaned from /src/parsing.c
├── 2D/                # 2D assets used in debug mode and for visual sfx
└── sounds/            # sfx / music files
```
